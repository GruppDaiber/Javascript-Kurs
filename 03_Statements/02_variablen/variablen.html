Javaskript: Variablen

Deklaration und Initialisierung

Javaskript-Variablen sind Container, um Daten oder Werte im Speicher des Programm (Computer) zur Verfügung zu halten.

Variablen können zur Programmlaufzeit beliebig oft beschrieben oder ausgelesen werden.

Variablen werden mit dem Schlüsselwort var deklariert (=im Programm angemeldet). Eine Initialisierung findet statt, wenn die deklariere Variable zum ersten mal mit einem Wert beschrieben wird.

Die "praktische" Initialisierung erfolgt über den Zuweisungsoperator "=" von rechts nach links in der Programmierzeile. 

						 				
<p>Ergebnis: <span id="ausgabe"></span></p>
<script>
var x;         // Deklaration 
			   // der Variablen x
x = 42;        // Initialisierung 
			   // der Variablen x
var y = 13;    // Gleichzeitige 
			   // Initialisierung und 
			   // Deklaration
			   // der Variablen y
var z = x + y; // Gleichzeitige 
			   // Initialisierung und 
			   // Deklaration
			   // der Variablen z über 
			   // eine Addition der Inhalte 
			   // von x und y  
document.getElementById('ausgabe').innerHTML = z;
</script>

Regeln im Umgang mit Variablen

  • Die Variablennamen müssen einzigartig benannt werden!
  • Die Variablennamen heissen im Englischen Identifiers.
  • Die Variablennamen sollten nach ihrer Tätigkeit beschreibend benannt werden. Der Quellcode wird dadurch erheblich verständlicher.
  • Die Variablennamen müssen folgendermaßen benannt werden:
    • Sie dürfen enthalten: Buchstaben, Ziffern, Unterlinie, Dollarzeichen
    • Der Variablenname muss mit einem Buchstaben beginnen.
    • Variablennamen dürfen auch mit $ oder _ beginnen.
    • Variablennamen unterscheiden sich durch die Groß-/Kleinschreibung (Case-Sensitiv): Otto, oTTo ... sind verschiedene Variablen!
    • Schlüsselwörter dürfen als Namen für Variablen nicht verwendet werden! 
						
/* Folgende Variablen werden 
   nur deklariert. D.h., die
   Inhalte der Variablen sind
   noch undefiniert!
*/   
var x;             /* */
var x_koordinate;  /* */
var wurzel;        /* */
var __Erdumfang;   /* */
var $_Patte;       /* */
var wurzel_2;      /* */

Deklaration/Initialisierung mehrerer Var. in einem Rutsch

Mehrere Variablen können auch in einer Zeile auf folgende Weise deklariert und initialisiert werden. 
						 
var person = "John Doe", carName = "Volvo", price = 200;
// oder:
var person = "John Doe", 
carName = "Volvo", 
price = 200;

// entspricht:

var person = "John Doe";
var carName = "Volvo";
var price = 200;

Strings und Numbers (=Zahlen)

Variablen können mit "Sätzen aus Buchstaben/Ziffern" oder mit Zahlen belegt werden. 
						 
var person = "John Doe"; // String
var ziffer = "4711";     // String
var coden_ist_toll = true;        // Boolean
var nicht_coden_ist_toll = false; // Boolean
var zahl = 42; 		// number -> Ganze Zahl (Integer)
var bruch = 3.14;  	/* number -> Fließkommazahl, oder Dezimalbruch(Float)*/

Arithmetik mit Variablen

Mit Variableninhalten kann gerechnet werden. Da in Javaskript die Typisierung der Variablen fehlt, müssen Sie auf die Variableninhalte achten! Vermeiden Sie unbedingt, dass eine Variable während des Programmablaufs nicht mit verschiedenen Typen von Inhalten belegt wird! Das ist problematisch beim Debuggen!

Addiere "5" + 2 + 3 ergibt einen String:

						 
var x = "5" + 2 + 3;
document.getElementById("erg1").innerHTML = x;

Addiere 5 + 2 + 3 ergibt eine Zahl:

						 
var x = 5 + 2 + 3;
document.getElementById("erg2").innerHTML = x;

Addiere "Hans " + "gehts " + "gut!" ergibt einen String:

						 
var x = "Hans " + "gehts " + "gut!";
document.getElementById("erg3").innerHTML = x;

Eingabe und Ausgabe mit Variablen

Variablen können beschrieben oder ausgelesen werden: 
						 
person = "Otto Walkes"; // Variable "person" wird beschrieben
komiker = person; /* Variable "person" wird ausgelesen und
                     der Wert ("Otto Walkes") in die Variable
                     "komiker" geschrieben*/
