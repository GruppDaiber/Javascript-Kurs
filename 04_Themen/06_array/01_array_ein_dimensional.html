Array: Viele Zahlen in einem eindimensionalen Array

  • Der Array ist nicht nur in Javaskript ein Objekt mit Eigenschaften und Methoden.
  • Der Array sammelt beliebig viele Elemente unter einem Namen zusammen.
  • Die einzelnen Element werden über Indizes angesprochen, um ausgelesen oder beschrieben werden zu können.
  • Die Element können Zahlen, Strings, boolsche Werte oder Objekte aufnehmen.

(0) Ein leerer Array wird zum Beispiel so angelegt (=Deklaration):
var zahlen_x = new Array();

(1) Der Array bekommt zwei zufällige Zahlen (Das sind Elemente des Arrays):
zahlen_x[0]=Math.floor(Math.random()*100); //erstes Element
zahlen_x[1]=Math.floor(Math.random()*100); //zweites Element

Alternativ zu (0) und (1):
var zahlen_x = new Array(Math.floor(Math.random()*100), Math.floor(Math.random()*100));

(2) Dem Array lassen sich automatisch über eine Schleife weitere Elemente anhängen:
for(i=2;i<20;i++){
zahlen_x[zahlen_x.length]=Math.floor(Math.random()*100);
}

(3) Ein beliebiges Element (z.B. das 4. Element) lässt sich über den Index auslesen:
document.getElementById("Ausgabe_3").innerHTML = zahlen_x[4];

(4) Ein beliebiges Element (z.B. das 4. Element) lässt sich auch neu beschreiben:
zahlen_x[4]="Polierte Platte mit 4711";
document.getElementById("Ausgabe_4").innerHTML =zahlen_x;
