String Eigenschaften

Eigenschaften Beschreibung Ausgabe
constructor Gibt die Construktorfunktion des Strings aus
length Gibt die Länge des Strings aus
prototype Erlaubt Eigenschaften und Methoden einem Objekt hinzuzufügen

String Methoden

Methoden Beschreibung Ausgabe
charAt() Gibt durch den spezifizierten Index das entsprechende Zeichen aus
charCodeAt() Gibt den Unicode des - durch den Index spezifizierten Zeichen - aus
concat() Verbindet mehrere Strings zu einem String zusammen
fromCharCode() Konvertiert Unicode-Zahlen zu den entsprechenden Zeichen
indexOf() Gibt den Index des im String spezifizierten Zeichens/String aus
lastIndexOf() Gibt den Index des im String zuletzt spezifizierten Zeichens/String aus
localeCompare() Vergleicht zwei Strings alphabetisch miteinander:
  • Ausgabe: -1 wenn str1 vor str2 kommt
  • Ausgabe: 0 wenn beide Strings gleich sind
  • Ausgabe: 1 wenn str1 nach dem str2 kommt
match() Durchsucht einen String nach einer Zeichenkette und listet alle Gefundenen auf:
replace() Durchsucht einen String nach einem spezifizierten Wert und ersetzt ihn mit einer Alternative.
search() Durchsucht einen String nach einem spezifizierten Wert und gibt die Position des gefundenen Wertes aus.
slice() Extrahiert einen Teil eines Strings und gibt den Rest wieder aus.
split() Zerlegt einen String nach spezifizierten Merkmalen und erstellt einen Array aus den Fragmenten.
substr() Extrahiert eine spezifizierte Anzahl von Zeichen aus einem String von einer spez. Position aus.
substring() Extrahiert Zeichen aus einem String zwischen zwei Positionen.
toLocaleLowerCase() Konvertiert einen String in Kleinbuchstaben (Abhängig von Spracheinstellung des Browsers)
toLocaleUpperCase() Konvertiert einen String in Großbuchstaben (Abhängig von Spracheinstellung des Browsers)
toLowerCase() Konvertiert einen String in Kleinbuchstaben
toUpperCase() Konvertiert einen String in Großbuchstaben
toString() Gibt den Wert eines String-Objekts zurück
trim() Entfernt am Anfang und Ende eines Strings die Whitespaces
valueOf() Gibt den primitiven Wert eines String-Objekts zurück
